# KRXZY Coin

A custom challenge coin featuring my online name, **KRXZY**, made for Hack Club.

## Design

The coin features:
- The letters **R X Z Y** stacked vertically on the left side
- A large "K" arrow/chevron as the centerpiece
- Two decorative stars framing the design
- A circular background

## How I Made It

1. **Figma** — I designed the coin's artwork in Figma, laying out the letters, the "K" arrow shape, and the star accents until I had a design I was happy with.
2. **SVG Export** — Once the design was finalized, I exported it from Figma as an SVG file.
3. **SVG → DXF Conversion** — I converted the SVG into a DXF file so it could be used for CAD modeling.
4. **Onshape** — I imported the DXF into Onshape and used it to model the actual 3D coin, giving it depth, thickness, and the physical form needed for manufacturing.

## Getting the Coin

This coin was submitted to **Hack Club**, and they'll be sending me a physical version of it as a 3D printed coin!

## Tools Used

- [Figma](https://www.figma.com/) — vector design
- SVG → DXF conversion
- [Onshape](https://www.onshape.com/) — CAD modeling
- [Hack Club](https://hackclub.com/) — coin production
